Don’t be hesitant when you make a decision

2.多和比你成功的人在一起和交往，人和人的区别就是看你如何处理问题
Work in an excellent environment

Be Responsible for yourself

8.面对自己不了解的事情，不要瞎猜测，而是去查 去问
If you don’t know just ask but not guess

Don’t try get something without hard work

Learn something smartly

人生由每一个小选择铸就
2021年02月05日 406 字 大概 1 分钟

人生是由每天的几个关键小抉择铸就

（1）起床-每天早上闹钟响的时候是否起床；
-【每天4点半起床，在所有人还在睡眠中时，我已经出发，开始元气满满的一天】
（2）规划-早上坐在电脑前面的第一个工作，是否选择先规划一下一天的生活；
- 【仿佛每时每刻都带着地图上图，任何时候都不会迷茫】
（3）行动- 开始做事儿之前是否选择尽量排除工作中可能受到的干扰；
- 【主动选择去减少生活中的干扰：开启番茄钟，戴耳塞，手机调成勿扰模式】
（4）整理- 晚上合上电脑之前，是否选择先整理一下桌面
-【让第二天准备开始工作的时刻，心情畅爽：清理电脑桌面和实体桌面】
（5）少看视频-每天想要拿起手机开视频的时候，时候能够用在重要的事情上，比如把看视频的时间腾挪出来给自己爱的人发微信。- 【心中有人，有梦，有烟火，不只让视频陪伴自己】
（6）遇到任何事情，第一反应不是拖着，而是选择想方设法去解决；
-【不懂的问题及时查/马上思考学习路线，一看就会，则立马解决；立马解决不掉，则设计学习路线，开始一点点学习；立马理解不了，写在 to do list 中/向别人求助/寻求合作】

每天关键瞬间的小选择 -> 好习惯 -> 大选择 -> 大成就

2021_aha_moment
2021年01月28日 701 字 大概 2 分钟

这里主要写下2021年所有启发，这是2021年最值得期待的时刻，也是自己成长的👣的见证

  1. 对小磕小碰的态度：虽不可避免，尽量小心和防患于未然；
    -【来源打篮球磕碰事件】

  2. 生活中难免有小磕小碰，事情虽然可能不大，但对正常的生活状态和心情影响挺大；
    -【换药6次*2h + 两次急诊 + 不能洗脸洗头】

  3. 做事儿做完美：走完99步，最后1步没走，也是失败；
    -【来源于《见识》/ 和买礼物事件（既然送礼，就别心疼，用心送到心里）】

  4. 每天看起来都很忙，但是做计划和不作计划还是有非常大的区别：一个是穷忙，一个是高效有产出的忙；

  5. 没有调理的信息一文不值，甚至会起到反作用
    -【所有网页标签归类了么？电脑文件夹有多乱，有多少无用信息？… …】

  6. 你以为你以为的就是你以为的？一定要常常问自己这句话，反思自己有太多的事儿是下意识就去做了，完全没有理性思考，一定要常常激发自己的慢思考；
    -【因为在第一天早上换的第一次药，之后就要每天都早上去么？； 《思考的快与慢》】

  7. 积累-思考每一个值得积累的事儿，认真积累起来，这不就是千里之行始于足下的道理么？反思过去，我到底扔了多少瓦砾。
    -【写到有用的函数，及时打包成R package；整理到有用的生物信息学 知识，及时整理到自己的R_for_Data_Science一书；整理有用的启发时刻，到自己的个人主页上；有用的网页标签做好整理】

  8. 严于律己，宽以待人，别较真；
    -【我今天有点明白较真的根本原因了，其实就是以自己的想法要求别人，给别人 灌输自己的想法，想要改变别人，想要别人按你的意愿做，事实上没个人都不同，做事的方法也没有对与错，只是对这件事的理解不一样而已，没法要求别人向自己一样做，所以能做的就是把自己的事情做好】

  9. 对于别人问问题的时候，无论问题怎么样，态度平和谦逊，不带任何情绪的解决问题，别评论别人问题的好坏，如果不想解决，直接推辞就好，不用类似落井下石似的说三道四 ；
    -【对于带硕士学习的启发，不能只与自己能力相当的人合作，要学会和能力比你差的人一起共处】
