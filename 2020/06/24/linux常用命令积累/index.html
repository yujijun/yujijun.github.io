linux常用命令积累

（1）常用基本命令
cp //拷贝
touch //新建空白文件
rm //删除文件
mkdir //新建文件夹
rmdir //删除文件夹
mv //移动文件
chown -R yjj:rstudo ./folder //更改文件的所属
（2） vim 打开文本之后，如何进行字符串搜索？
: /test 将定位到test位置，然后用n来进行跳转到不同的test行。

（3）Linux文件权限设置
chmod u|g|o|a =(设置权限)|+(添加权限)|-(减少权限) r|w|x 文件或者目录
for example:
chmod u=rwx file //u example user
